# Repository Overview

This project is a Flutter-based social media application. It includes authentication, home feed, post creation, and profile management features. Firebase services such as Authentication, Firestore, and Storage are integrated across the app.

## Key Directories

- **lib/config**: App-wide configuration, including Firebase options.
- **lib/features**: Feature-based modules (`auth`, `home`, `post`, `profile`, `storage`).
- **lib/themes**: Theme definitions (e.g., light mode).
- **test**: Dart/Flutter testing suite.
